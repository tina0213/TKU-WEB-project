# TKU-WEB-project
* **主要目的：**
  * 提升淡江學生選擇早餐時的便利性。
  * 促進店家間的良性競爭、問題改善。
* **想要解決的問題：**  
  * 學校周邊早餐店沒有專屬公開的討論平台。
  * 各種資訊缺乏統整。
  * 店家無法得知學生用餐後的回饋。
  * 用餐時間人潮眾多，現場等待需花費大量時間。
* **功能：**  
  * 提供各店家的簡介、電話、菜單以及位置以供快速查詢。
  * 留言板功能。可針對特定店家留下評語及評分。
* **開發工具：**    
  * 開發平台：visual studio2019---ASP.NET MVC
  * UML 模型：draw.io
  * 資料庫：SQL Sever
